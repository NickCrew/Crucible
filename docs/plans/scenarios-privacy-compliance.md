# Privacy & Compliance Scenarios (GDPR/CCPA Focus)

- **PII Harvesting via Enumeration**: A slow, "low and slow" attack that crawls user profile endpoints specifically looking for sensitive fields like SSNs, credit card fragments, or healthcare IDs.
- **Right-to-be-Forgotten Bypass**: Testing if a user's data is still accessible via a "cached" or "legacy" API version after they have requested account deletion.
